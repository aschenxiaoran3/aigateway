# ai-rules

`ai-rules/` 是 Q2 阶段 `ai-platform` 的唯一规则源。

目录约定：

- `prompts/`: Agent 级 Prompt 模板
- `rules/`: 面向角色和治理口径的规则文档
- `skills/`: 可移植技能包说明
- `contracts/`: 节点输入输出契约
- `pipelines/`: 官方流水线模板

使用原则：

1. 所有控制平面对象优先引用本目录中的文件，而不是散落在仓库其它位置的旧路径。
2. 数据库只存索引、发布状态和运行记录；真正的规则内容以文件为准。
3. 新增规则时优先复用既有文件命名与版本策略，避免同义重复。
